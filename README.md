# Verso

A modern rewrite of Redfin's [React Server](https://github.com/redfin/react-server) – performant server rendering via an isomorphic multi-root architecture, usable out of the box with Vite.

## Status

Pre-alpha. This is something I'm building for fun, and because I want to be able to experience the joy of React Server again. I don't have any illusions of this being used for anything serious, but I do want it to, you know, work.
